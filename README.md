# 🎥 VideoHub99 - Real-Time Video Sharing Platform

একটি আধুনিক, রিয়েল-টাইম ভিডিও শেয়ারিং প্ল্যাটফর্ম যা **YouTube, Google Drive, Vimeo, Dailymotion**, এবং সরাসরি **MP4 ফাইল** সাপোর্ট করে।

## ✨ ফিচার

- 🎬 **মাল্টি-প্ল্যাটফর্ম ভিডিও সাপোর্ট** (YouTube, Drive, Vimeo, DailyMotion, Direct MP4)
- 🔴 **রিয়েল-টাইম আপডেট** (Socket.io এর মাধ্যমে)
- 👥 **ব্যবহারকারী ম্যানেজমেন্ট** (Admin প্যানেলসহ)
- ❤️ **লাইক এবং সেভ সিস্টেম**
- 🔍 **উন্নত সার্চ ফিচার**
- 📱 **মোবাইল-রেডি ডিজাইন**
- 🚀 **Express.js + Socket.io সার্ভার**
- 🎨 **সুন্দর UI/UX**

## 🚀 দ্রুত স্টার্ট

### প্রয়োজনীয়তা
- Node.js ১৬+
- npm বা yarn

### ইনস্টলেশন

```bash
# ডিরেক্টরিতে যান
cd Videohub99

# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# সার্ভার চালু করুন
npm start
```

সার্ভার `http://localhost:5000` এ চলবে।

## 🔌 API এন্ডপয়েন্ট

| মেথড | এন্ডপয়েন্ট | বর্ণনা |
|-------|------------|--------|
| GET | `/api/videos` | সব ভিডিও পান |
| POST | `/api/videos` | নতুন ভিডিও যোগ করুন |
| POST | `/api/videos/:id/like` | ভিডিও লাইক করুন |
| DELETE | `/api/videos/:id` | ভিডিও মুছুন |
| POST | `/api/users/register` | নতুন ব্যবহারকারী |
| GET | `/api/users` | সব ব্যবহারকারী |

## 🔄 রিয়েল-টাইম ইভেন্ট

Socket.io ইভেন্টগুলি স্বয়ংক্রিয়ভাবে ফ্রন্টএন্ডকে আপডেট করে:

- `video:new` - নতুন ভিডিও এসেছে
- `video:deleted` - ভিডিও মুছা হয়েছে
- `video:like-updated` - লাইক আপডেট হয়েছে
- `user:registered` - নতুন ব্যবহারকারী
