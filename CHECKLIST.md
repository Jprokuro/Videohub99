## 🐛 ট্রাবলশুটিং

### সার্ভার শুরু হচ্ছে না?
```bash
# পোর্ট ১৬০০ ব্যবহার করে দেখুন
PORT=1600 npm start
```

### Node modules এরর?
```bash
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
```

### Socket.io কানেক্ট হচ্ছে না?
- ব্রাউজার কনসোল খুলুন (F12)
- Network ট্যাব চেক করুন
- 5000 পোর্ট চলছে কিনা দেখুন
